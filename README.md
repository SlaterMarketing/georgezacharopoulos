# michaeljbevan.com

Site for **Michael Bevan** — Astro 5 + `@kintana/sdk` on Cloudflare Pages.

André De Freitas–style layout: full-screen hero, bottom nav, dark + yellow brand.

Copy `.env.example` to `.env`, add Kintana credentials, then:

```bash
npm install
npm run dev
```

**Deploy:**

```bash
npm run deploy
```
